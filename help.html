How to use the Join app
  1. Create a task on the "Add Task" page
  2. On following "Backlog" page you can find and edit the created tasks
  3. Click on a task field to edit it. Here you have the possibilites to move it to the "Board" page, to edit it or to delete it
  4. The tasks sent from the "Board" page are displayed first in the "To Do" column and can be moved with drag & drop depending on the processing status.
  5. Click on the task to edit it. Here you have the possibilites to send it to the "Board" page, to edit it or to delete it too
